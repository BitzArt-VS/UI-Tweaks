A note on development

UI Tweaks is AI-assisted.

Some of the code in this mod was written with the help of AI tools — but this is not a "vibe-coded" project. Design, architecture, and testing are handled by a human, and a meaningful share of the code is hand-written as well; AI is used to execute well-defined coding tasks under direction. Every change is reviewed and verified before it lands in the codebase, and nothing reaches a release unless it meets the quality bar set for how the mod should look, feel, and behave in your game. If something falls short of that, please report it.

UI Tweaks respects vanilla game

All major features are optional and disabled by default, so you can choose exactly what to enable and how it should behave. See the Configuration section below to turn on the features you want and adjust them to your liking.

Configuration

Press Shift + N (default) in-game to configure UI Tweaks.

Mod configuration file is located at ModConfig/ui-tweaks.json, generated automatically on first launch. See Configuration for full reference.

Features
Zoom Button💡 Inspired by Zoom Button.

Enable Zoom, restart the game, then hold Z to zoom the camera in for a closer look at distant blocks, animals, ruins, and suspicious shapes on the horizon.

For locating cattails, clay, ruins, and every other thing you swear was right there a second ago.

HUD

Environment Widget

A small overlay showing the in-game date and time, current temperature, and your coordinates. Position, size, format, and background opacity are all configurable.

Because “I should check the character menu” is how bears get closer.

Status Tooltips 💡 Inspired by Extended HUD Info.

Numeric readouts on your HUD bars:

  • HP — current / max health above the health bar
  • Satiety — current / max satiety above the satiety bar
  • Hunger rate — your current hunger drain rate
  • Temporal Stability — personal stability and location stability

Each tooltip can be toggled and repositioned independently.

If your character is cold, wet, hungry, temporally unstable, and somehow still carrying twelve stacks of stones, the least the HUD can do is be specific.

Custom Tooltips

You can define your own HUD tooltip elements from the config file — choose what to display, where to place it, and how to format it. Supports VTML and variable placeholders.

If the game tracks it, someone will eventually want it pinned to the screen.

Quick Search & Calculator💡 Inspired by Satisfactory's quick-search.

Press N (configurable) to search handbook entries by name without opening the handbook. Results filter as you type.

Also evaluates arithmetic expressions (+, -, *, /) directly in the search bar. Useful when you don't want to resort to an external calculator.

For when the crucible wants percentages and your brain has already left the room.

Game Tweaks

Various tweaks applied to the vanilla game:

  • Calendar year — years now start at 1 instead of 0 (display only, no effect on in-game logic)

A harmless little oddity, now safely overreacted to.
