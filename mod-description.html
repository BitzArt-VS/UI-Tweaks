Ever eaten a bowl of stew only to be left with a 1% serving you can't eat and can't stack?

UI Tweaks slaps the hard numbers directly on your HUD so you can actually min-max your bowl of stew.

Features
HUD

Environment Widget

A small overlay showing the in-game date and time, current temperature, and your coordinates. Position, size, format, and background opacity are all configurable.

Status Tooltips 💡 Inspired by Extended HUD Info.

Numeric readouts on your HUD bars:

  • HP — current / max health above the health bar
  • Satiety — current / max satiety above the satiety bar
  • Hunger rate — your current hunger drain rate
  • Temporal Stability — personal stability and location stability

Each tooltip can be toggled and repositioned independently.

Custom Tooltips

You can define your own HUD tooltip elements from the config file — choose what to display, where to place it, and how to format it. Supports VTML and variable placeholders.

Quick Search & Calculator💡 Inspired by Satisfactory's quick-search.

Press N (configurable) to search handbook entries by name without opening the handbook. Results filter as you type.

Also evaluates arithmetic expressions (+, -, *, /) directly in the search bar. Useful when you don't want to resort to an external calculator.

Game Tweaks

Various tweaks applied to the vanilla game:

  • Calendar year — years now start at 1 instead of 0 (display only, no effect on in-game logic)
Configuration

Config file is generated automatically on first launch at ModConfig/ui-tweaks.json.

See Configuration for full reference.

Compatibility
Status Mod Comment
