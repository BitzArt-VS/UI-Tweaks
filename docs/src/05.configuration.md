## Mod configuration

The mod can be configured by editing the `ui-tweaks.json` file in your Vintage Story's `ModConfig` folder.

Keep in mind that changing some of these options may require you to restart the game or reload the world for the changes to take effect.

> **Note**: You need to start a game with this mod installed at least once for it to generate the default configuration file.

## Configuration values

<details>
<summary><code>hud</code> — Configuration for the heads-up display (HUD) features of the mod.</summary>

<div style="padding-left: 1.5em;">

<details>
<summary><code>healthbarTooltip</code> — Configuration for the health bar tooltip.</summary>

<div style="padding-left: 1.5em;">

- `enable`: Whether to show the healthbar tooltip. Default: `true`.

- `dialogArea`: The dialog area anchor for the tooltip. Default: `CenterBottom`.

- `height`: Height of the tooltip in pixels. Default: `25.0`.

- `width`: Width of the tooltip in pixels. Default: `144.0`.

- `centerText`: Whether to center the tooltip text. Default: `false`.

- `offset`: Offset of the tooltip from its default position.

<div style="padding-left: 1.5em;">

- `x`: Horizontal offset. Default: `-250.0`.

- `y`: Vertical offset. Default: `-88.0`.

</div>

- `hasBackground`: Whether to show a background behind the tooltip. Default: `true`.

- `backgroundOpacity`: Opacity of the background. Default: `0.5`.

- `backgroundCornerRadius`: Corner radius of the background. Default: `12.0`.

- `format`: Format of the tooltip text. Supports [VTML](https://wiki.vintagestory.at/VTML) and [variable placeholders](#variable-placeholders). Default: `<font align=left>{player-health-current} / {player-health-max}</font><font align=right>({player-health-percent}%)</font>`.

- `fontSize`: Font size of the tooltip text in pixels. Default: `14.0`.

</div>

</details>

## Variable placeholders

These placeholders can be used in any tooltip format string:

<details>
<summary>Player Health</summary>

<div style="padding-left: 1.5em;">

- `{player-health-current}` - The player's current health.

- `{player-health-max}` - The player's maximum health.

- `{player-health-percent}` - The player's current health as a percentage of maximum health.

</div>

</details>

<details>
<summary>Player Satiety</summary>

<div style="padding-left: 1.5em;">

- `{player-satiety-current}` - The player's current satiety.

- `{player-satiety-max}` - The player's maximum satiety.

- `{player-satiety-percent}` - The player's current satiety as a percentage of maximum satiety.

- `{player-satiety-hunger}` - The player's current hunger rate.

</div>

</details>

<details>
<summary>Temporal Stability</summary>

<div style="padding-left: 1.5em;">

- `{player-temporal-stability}` - The player's current temporal stability level.

- `{player-location-temporal-stability}` - Temporal stability at the player's position.

</div>

</details>
