## Mod configuration

The mod can be configured by editing the `ui-tweaks.json` file in your Vintage Story's `ModConfig` folder.

Keep in mind that changing some of these options may require you to restart the game or reload the world for the changes to take effect.

> **Note**: You need to start a game with this mod installed at least once for it to generate the default configuration file.

### Available options

#### HUD

**HealthbarTooltip**: Configuration for the health bar tooltip.

  - `enable`: Whether to show the health bar tooltip. Default: `true`.
  - `format`: The format of the tooltip text. Default: `{current} / {maximum} ({percent}%)`. You can use the following placeholders in your format string:
    - `{current}` (or `{curr}`, `{cur}`, `{c}`, `{0}`): Current health.
    - `{maximum}` (or `{max}`, `{m}`, `{total}`, `{1}`): Maximum health.
    - `{percentage}` (or `{percent}`, `{p}`, `{%}`, `{2}`): Current health as a percentage of maximum health.
  - `offset`: The offset of the tooltip from its default position.
    - `x`: Horizontal offset.
    - `y`: Vertical offset.

**SatietyTooltip**: Configuration for the satiety tooltip.

  - `enable`: Whether to show the satiety tooltip. Default: `true`.
  - `format`: The format of the tooltip text. Default: `{current} / {maximum} ({percent}%)   |   {hunger}%`. You can use the following placeholders in your format string:
    - `{current}` (or `{curr}`, `{cur}`, `{c}`, `{0}`): Current satiety.
    - `{maximum}` (or `{max}`, `{m}`, `{total}`, `{1}`): Maximum satiety.
    - `{percentage}` (or `{percent}`, `{p}`, `{%}`, `{2}`): Current satiety as a percentage of maximum satiety.
    - `{hunger}` (or `{h}`, `{3}`): Current hunger rate.
  - `offset`: The offset of the tooltip from its default position.
    - `x`: Horizontal offset.
    - `y`: Vertical offset.

#### Quick Search

- `enable`: Whether to enable the quick search feature. Default: `true`.
- `resultListHeight`: The height of the quick search result list in pixels. Default: `200`.
