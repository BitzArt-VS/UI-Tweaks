# Installation

## From official mod repository

You can install UI Tweaks from ModDB using this link: [UI Tweaks on ModDB](https://mods.vintagestory.at/uitweaks).

## From GitHub

You can download the latest release of UI Tweaks from the [releases page](https://github.com/BitzArt-VS/UI-Tweaks/releases). Simply download the latest `.zip` file and place it in your `Mods` folder.
